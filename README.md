# Система анализа поступления
Командный кейс №3 «Анализ поступления»  
Московская предпрофессиональная олимпиада школьников, профиль «Информационные технологии»

## Описание
Десктоп-приложение для автоматизации работы приёмной комиссии вуза. Система анализирует динамику конкурсных ситуаций при поступлении абитуриентов с учётом приоритетов заявлений, согласий о зачислении и изменения конкурсных списков в ходе приёмной кампании.

## Ключевые возможности
- Загрузка и инкрементальное обновление конкурсных списков (время обработки ≤ 5 секунд)
- Расчёт проходных баллов с корректным учётом приоритетов заявлений и эффекта вытеснения
- Визуализация данных с поддержкой фильтрации и сортировки (время отклика ≤ 3 секунды)
- Автоматическая генерация тестовых данных с точным соблюдением условий пересечений (п. 2.8–2.9 технического задания)
- Формирование PDF-отчётов со статистикой поступления

## Требования к окружению
- Python 3.8 или выше
- Библиотеки, перечисленные в файле requirements.txt
- Для запуска скомпилированной версии: операционная система Windows 10/11

## Установка и запуск

### Запуск из исходного кода
1. Создайте виртуальное окружение в директории с файлом main.py:
   ```
   python -m venv venv
   ```
2. Активируйте виртуальное окружение:
   - Windows: `venv\Scripts\activate`
   - Linux/macOS: `source venv/bin/activate`
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
4. Запустите приложение:
   ```
   python main.py
   ```

### Запуск скомпилированной версии
1. Перейдите в директорию `binary`
2. Запустите файл `main.exe`

Скомпилированная версия не требует дополнительной установки и готова к работе после запуска.
